# csgo-simple-panorama
With the release of CS2, Valve discontinued support for CSGO, leaving many CSGO interface elements redundant and non-functional. This panorama mod removes all unsupported features from the interface.

## How to install?
* Download the csgo-simple-panorama zip file from releases tab and extract it where csgo.exe is
* Replace panorama.dll inside of the bin folder where csgo.exe is for this to work! modified panorama.dll is in the latest release.
* In the CSGO launch options on Steam, set the launch option "-game csgosimple/csgo -nosecure"
* If you did everything correctly, you will see a clean interface without any unnecessary clutter.
* That's it, enjoy the clean CSGO interface!
